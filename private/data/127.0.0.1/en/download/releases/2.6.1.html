Python 2.6.1

Python 2.6.1 has been replaced by a newer bugfix release of Python. Please download Python 2.6.6 instead.

Python 2.6.1 was released on December 4th, 2008.

This is the first bugfix release of Python 2.6. Python 2.6 is now in bugfix-only mode; no new features are being added. Dozens of bugs that were reported since the release of 2.6 final have been fixed.

See the detailed release notes for more details.

  • Andrew Kuchling's guide to What's New in Python 2.6.
  • NEWS file contains a listing of everything that's new in each alpha, beta, and release candidate of Python 2.6.
  • PEP 361.

Please report bugs at http://bugs.python.org

See also the license.
